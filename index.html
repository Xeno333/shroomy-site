Welcom to Shroomy!

Shroomy is a 2D game engine written in C++ for making games in Lua. Shroomy has an intigrated mod loading system.


Shroomy can be donwloaded on GitHub.


Supported platforms:

  • Linux

Features:

  • Lua-defined Client/Server interfaces
  • Fully Lua-defined games and mods
  • Custom inputs through SDL
  • Core API functions
  • Basic entities and collision logic
  • Support for .wav audio assets
  • Support for .png visual assets
